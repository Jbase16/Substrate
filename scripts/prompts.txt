# Benchmark prompts: 50 prompts spanning five categories.
# Lines starting with '#' are comments and skipped by the harness.
# Each non-comment, non-blank line is one prompt. Order is shuffled deterministically
# at run time using a fixed seed so categories don't cluster runtime-stat patterns.

# === Category 1: Technical / systems (10) ===
Explain why SSD streaming is the bottleneck for local LLM inference.
Describe the difference between attention and feed-forward layers in a transformer.
What is KV cache and why does it matter for autoregressive generation?
Explain how rotary position embeddings work in modern language models.
Why does mixed-precision quantization preserve quality better than uniform quantization?
What are the main differences between INT8 and FP8 in deep learning hardware?
Explain how speculative decoding accelerates language model inference.
Describe how memory bandwidth limits throughput on Apple Silicon Macs.
What is the role of group_size in weight quantization schemes like GPTQ?
Explain how flash attention reduces memory consumption during training.

# === Category 2: Reasoning / step-by-step (10) ===
A train leaves Boston at 3pm going 60 mph. Another leaves NYC at 4pm going 80 mph toward Boston. They are 220 miles apart. When do they meet?
If a shirt is on sale for 30 percent off and then another 20 percent off the discounted price, what is the total discount from the original price?
Sarah has three brothers. Each brother has two sisters. How many sisters does Sarah have?
A bat and ball cost 1.10 together. The bat costs 1.00 more than the ball. How much does the ball cost?
If it takes 5 machines 5 minutes to make 5 widgets, how long does it take 100 machines to make 100 widgets?
List the steps to safely change a flat tire on the side of a highway.
Explain how to debug a memory leak in a long-running Python process.
Walk through how to set up a Postgres replica with streaming replication.
Describe the steps to migrate a Django project from SQLite to Postgres.
Explain why the halting problem is undecidable, in terms a programmer would find concrete.
